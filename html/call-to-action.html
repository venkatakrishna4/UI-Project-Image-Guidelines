Call-To-Action (CTA)

The image contains call-to-action elements that mimic an interactive component like a button.

1. Call to action text in combination with a visual element:

  • Button-like, that appears clickable to the customers in the first degree source
  • IUnderlined call to action text in the first degree source
  • Call to action with an arrow
  • Visual element combined with text which may lead to interaction with other sources

2. URL or phone number combined with visual element, e.g. search bar, or underlined without regard to a visual element at the first degree source

3. QR Codes: available in the first degree source

4. Commonly used interactive icons: that appears clickable like heart-, share-, or download-buttons

Exception:

  • Call to action text that is not presented as a button or part of any other visual element and Button on natural settings such as on a screen or part of a screenshot is fine to approve.
  • Search bars are not considered as interactive as long as they contain words and have no call to action or URL
  • QR code apart of natural setting is fine to approve (i.e. in second degree source) CTA Type

What is the action that the CTA text conveys?

  1. Download, Install, Get the app
  2. Learn More
  3. Play Now, Start Game, Enter Game Try New Character
  4. Order Now, Try Now, Buy Now, Shop Now, Subscribe Now, Join Now
  5. Something else not listed above
